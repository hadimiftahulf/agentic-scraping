# Sprint 0: Foundation & Monorepo Setup

**Goal:** Establish the monorepo skeleton and shared infrastructure.

## Tasks
- **T-001**: Initialize Monorepo structure (apps/, packages/, infra/).
- **T-002**: Setup Node.js Workspaces and TypeScript base config.
- **T-003**: Define Prisma Schema and initialize PostgreSQL.
- **T-004**: Configure Docker Compose for local development (DB, Redis).
- **T-005**: Setup Centralized Environment Management (Zod).
- **T-006**: Initialize Shared Utils package.

## Definition of Done
- `docker-compose up` runs successfully with healthy containers.
- Prisma migration can be applied.
- All service placeholders return a health status.
