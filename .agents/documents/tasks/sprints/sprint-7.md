# Sprint 7: Multi-Account & Analytics

**Goal:** Scale the system to support multiple Facebook accounts and provide business insights.

## Tasks
- **T-701**: Refactor Worker to support concurrent sessions for multiple accounts.
- **T-702**: Update Database Schema to include `Account` entity.
- **T-703**: Implement Account Management UI in Dashboard.
- **T-704**: Build Analytics Dashboard (Post volume, Success rate, Sales tracking).
- **T-705**: Implement CSV Export for product data and posting history.

## Definition of Done
- Multiple Facebook accounts can be configured and used for posting.
- Dashboard shows aggregated stats across all accounts.
- Data can be exported for external analysis.
