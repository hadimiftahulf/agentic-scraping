# Sprint 4: Frontend Dashboard (Next.js)

**Goal:** User interface for monitoring products, controlling the bot, and viewing logs.

## Tasks
- **T-401**: Setup Next.js 14 project in `apps/web`.
- **T-402**: Design and implement dark-themed Design System.
- **T-403**: Implement API Client with React Query.
- **T-404**: Build Product List View with status filters and search.
- **T-405**: Build Product Detail Drawer with job history logs.
- **T-406**: Implement "Post" button with Optimistic UI updates.
- **T-407**: Implement Real-time Status Polling for processing items.
- **T-408**: Build Settings Page for system configuration.

## Definition of Done
- Dashboard displays products from the database correctly.
- Post action triggers the backend and updates status in real-time.
- Settings changes are persisted and applied immediately.
- UI is responsive on mobile and desktop.
